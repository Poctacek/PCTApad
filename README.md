# PCTApad
## A 3-button basic hackpad by PCTA

I followed the hackpad [guide](https://hackpad.hackclub.com/guide) to make this.

It's my first PCB project and its very basic, but i thought it was cool to make and I definitely learned something new about 3D modeling and pcb stuff.

<u>BOM</u>:

- 1x Custom PCB (FR-4)

- 1x 3D Printed Bottom Case (PLA)

- 1x 3D Printed Top Housing (PETG/PLA)

- 3x Red mechanical switches

- 3x PBT Keycaps

- 4x M3 Screws / Heatset inserts

- 1x USB-C connector (fem)
